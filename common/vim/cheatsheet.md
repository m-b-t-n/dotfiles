# My Cheat Sheet

## Tab（タブ）

| cmd    | description        |
| tabnew | タブを新規に開く   |
| gt     | 次のタブに移動する |
| gT     | 前のタブに移動する |

## Searching（検索）

| cmd   | description                        |
| :nohl | 検索した単語のハイライトを解除する |

## Binary（バイナリ）

## Pane（ペイン）

| cmd    | description        |

### winresizer

| <C-e>   | リサイズモードに入る                         |
| <Enter> | リサイズを確定する                           |
| <Esc>   | リサイズモードから抜ける（変更は破棄される） |
| h       | 現在のペインを左に拡大する                   |
| j       | 現在のペインを下に拡大する                   |
| k       | 現在のペインを上に拡大する                   |
| l       | 現在のペインを右に拡大する                   |
